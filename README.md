Pac-Man Game 
Overview
This project recreates the classic Pac-Man game using Java Swing and AWT, with backend integration to MySQL via JDBC. Players enjoy arcade-style gameplay, and their scores are saved into a database for leaderboard tracking and performance analysis.

Features
Classic Pac-Man gameplay with movement and collision detection

Popup for player name entry at game-over

Automatic score saving into MySQL database

Leaderboard queries for top scores and player statistics

Technologies Used
Java (JDK) – Game logic and GUI

VS Code – Development environment

MySQL – Database for score storage

MySQL Connector/J – JDBC driver for database connectivity
